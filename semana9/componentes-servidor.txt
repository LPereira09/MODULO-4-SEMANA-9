Sistema operativo:
Es el software que administra el hardware y los recursos del equipo.
Permite ejecutar programas y mantener el servidor en funcionamiento.

Node.js:
Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador.
Se utiliza para desarrollar y ejecutar aplicaciones del lado del servidor.

Aplicación server.js:
Es el archivo principal donde se encuentra la lógica del servidor.
Define las rutas y las respuestas que recibirán los usuarios.

Puerto:
Es el punto de comunicación por donde el servidor recibe las solicitudes.
Permite que los clientes se conecten al servicio.

Logs:
Son registros que almacenan la actividad y los eventos del servidor.
Ayudan a detectar errores y monitorear el funcionamiento de la aplicación.
